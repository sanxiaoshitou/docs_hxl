## 一、Claude Code
Claude Code (CLI 工具) ：本指南的核心主题。这是一款专为开发者设计的、基于终端的代理工具，用于深度代码交互和自动化任务。   

claude.ai (网页版) ：一个通用的、基于网页的聊天机器人。尽管它也具备强大的代码生成能力，但其交互模型与 Claude Code 完全不同，更侧重于对话式问答和内容生成。  

Claude for Desktop (桌面应用) ：一款适用于 macOS 和 Windows 的独立桌面应用程序，提供持久化的工作流体验。它通过模型上下文协议（Model Context Protocol, MCP）
服务器拥有自己的扩展机制，但与原生于终端的 Claude Code 是两个不同的产品。目前，Linux 尚不支持该桌面应用。
